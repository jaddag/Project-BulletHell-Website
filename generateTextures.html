Class: generateTexture

This class provides methods to generate Texture and Pixmap objects using a grid system and a list of Color objects. It is used to create pixel-based graphical assets for the game.

Method: genTexture()

Generates a LibGDX Texture from a 2D grid of colors.

  • Parameters:
    • gridSizeX, gridSizeY: Dimensions of the grid
    • pixelSize: Size of each pixel block
    • arrayList: List of colors used for each pixel
  • Returns: A Texture object created from the colored pixel grid.
  • Automatically disposes of the Pixmap after creating the Texture.

Method: genPixmap()

Similar to genTexture(), but returns the raw Pixmap instead of creating a Texture.

  • Useful when further processing or saving the Pixmap is needed.
  • The caller is responsible for disposing of the returned Pixmap.
