Method: loadAssets()

This method generates and saves graphical assets for the game if the genEverytimeFlag is set to true. These assets are written as PNG files into the local cache directory.

Assets Created

  • background1.png – dense star background
  • background2.png – medium star background
  • background3.png – sparse star background
  • starBackground.png – special starfield texture
  • alphaBG.png – semi-transparent black overlay
  • attackGrid.png – glowing red grid texture
  • attackGridREACT.png – reactive variant of the grid
  • attackRing.png – glowing red ring texture

Notes

  • Uses helper classes bgGenerator and generateGlowTextures
  • Randomized values for varied star patterns
  • All created Pixmaps are disposed after writing to save memory
