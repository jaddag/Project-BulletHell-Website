Methode: loadAssets()

Diese Methode erstellt und speichert grafische Assets für das Spiel, wenn das Flag genEverytimeFlag auf true gesetzt ist. Die Assets werden als PNG-Dateien im lokalen cache-Verzeichnis abgelegt.

Erstellte Assets

  • background1.png – dichter Sternenhintergrund
  • background2.png – mitteldichter Sternenhintergrund
  • background3.png – spärlicher Sternenhintergrund
  • starBackground.png – spezielle Sternfeldtextur
  • alphaBG.png – halbtransparenter schwarzer Overlay
  • attackGrid.png – leuchtende rote Gittertextur
  • attackGridREACT.png – reaktive Variante des Gitters
  • attackRing.png – leuchtende rote Ringtextur

Hinweise

  • Verwendet die Hilfsklassen bgGenerator und generateGlowTextures
  • Verwendet Zufallswerte für abwechslungsreiche Sternenmuster
  • Alle erstellten Pixmaps werden nach dem Speichern freigegeben, um Speicher zu sparen

Codeausschnitt

public void loadAssets() {
    if (genEverytimeFlag) {
        // Pixmap-Assets erstellen und als PNG-Dateien speichern
        ...
    }
}
